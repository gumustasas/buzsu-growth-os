# Knowledge — Damıtılmış Dış Araştırma Notları

Bu klasör `knowledge-graph/`'tan **farklıdır**: `knowledge-graph/` BUZSU/Suvesu'nun entity
verisini (ürün, marka, kirletici, teknoloji) tutar; `knowledge/` ise dış repo araştırmasından
çıkarılan, alan-bağımsız (cross-cutting) desenleri ve karşılaştırmaları tutar. Skill modülleri
bu notlara referans verebilir; entity verisi burada tutulmaz.

## Dosyalar

- `geo-scoring-frameworks.md` — 3 farklı GEO skorlama modelinin karşılaştırması
- `n8n-workflow-process.md` — n8n-mcp'nin dokümantasyon-önce iş akışı
- `security-domains-map.md` — güvenlik alanlarının BUZSU platformuna eşlenmesi
- `skill-format-conventions.md` — Claude Code skill dosya formatı kuralları
- `memory-architecture-patterns.md` — oturumlar arası bağlam/hafıza mimarisi desenleri
