# Playbooks — Çapraz Alan Senaryoları

Tek bir skill modülünün değil, birden fazla agent/skill'in sırayla devreye girdiği uçtan uca
BUZSU senaryoları. Her playbook hangi skill'lerin/agent'ların hangi sırayla çalıştığını ve
hangi adımda insan onayı gerektiğini belirtir.

## Dosyalar

- `yeni-urun-sayfasi-launch.md` — schema + seo + cro + ui-ux
- `aylik-geo-audit.md` — geo + aeo-ai-search + entity-seo + content
- `n8n-otomasyon-kurulum.md` — n8n + mcp + github
- `guvenlik-denetimi.md` — security + php + codeigniter4
- `whatsapp-satis-optimizasyonu.md` — cro + ecommerce + content
