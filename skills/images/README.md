# Görsel Optimizasyon

## Amaç

Ürün görsellerinin SEO/performans/erişilebilirlik açısından doğru yapılandırılması.

## İlgili Agent

Yok — content-agent ve cro-agent çıktılarına destek sağlar.

## İncelenen Dış Kaynaklar

- **ComposioHQ/awesome-claude-skills** — Katalogda 'image enhancement' kategorisinde bir skill bahsi tespit edildi; içerik derinlemesine incelenmedi (yüzeysel referans). Bu modülün pratikleri genel web performansı/SEO bilgisine dayanır, dış repodan doğrudan alıntı yapılmamıştır.

> Kör kopyalama yapılmadı; yalnızca BUZSU bağlamına uyarlanabilir pratik/desen çıkarıldı. Detaylı atıf için bkz. kök dizindeki `reports/external-repo-integration-report.md`.

## Bu Modüldeki Dosyalar

- `skill.md` — Claude Code skill tanımı (ne zaman devreye girer, sorumluluklar, iş akışı)
- `checklist.md` — Uygulama/denetim kontrol listesi
- `prompts.md` — Örnek görev promptları (Türkçe)
- `implementation.md` — BUZSU'ya özel uygulama adımları ve sınırlar

## Onay Notu

Bu modül için dış kaynak derinlemesine incelenmedi — sonraki bir oturumda ComposioHQ kataloğundaki ilgili skill ayrıca taranabilir (rapor 'Sonraki Yapılacaklar' bölümünde not edildi).
