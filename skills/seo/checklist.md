# SEO — Arama Motoru Optimizasyonu — Kontrol Listesi

- [ ] Hedef anahtar kelimenin arama hacmi ve niyeti belirlendi mi?
- [ ] Mevcut sıralama Serper/GSC verisiyle doğrulandı mı (kör tahmin yok)?
- [ ] Title etiketi 50-60 karakter, description 140-160 karakter aralığında mı?
- [ ] Canonical URL doğru sayfayı işaret ediyor mu?
- [ ] Hreflang (varsa çok dilli sayfalar için) doğru yapılandırılmış mı?
- [ ] H1-H2-H3 hiyerarşisi anahtar kelimeyi doğal şekilde içeriyor mu?
- [ ] Sayfa Core Web Vitals eşiklerini (LCP < 2.5s, CLS < 0.1, INP < 200ms) karşılıyor mu?
- [ ] İç bağlantı fırsatları (Suvesu makalesi → Buzsu ürün sayfası) listelendi mi?
- [ ] Rakip SERP'te öne çıkan snippet/PAA fırsatı var mı (competitor-agent'a görev gerekir mi)?
- [ ] Schema markup eksikliği varsa schema-agent'a görev üretildi mi?
- [ ] Search Console tıklama/gösterim verisi yorumlandı mı (insan veri sağladıysa)?
- [ ] Öneri kör değil; SERP + Serper snapshot'ına dayanıyor mu? (CLAUDE.md madde 4)

## Genel BUZSU Kuralları (her zaman geçerli)

- [ ] Çıktı önce `/drafts` altına yazıldı (CLAUDE.md madde 2)
- [ ] Ürün/fiyat/sağlık iddiası abartılmadı, doğrulanabilir (CLAUDE.md madde 5)
- [ ] PII (telefon, isim, müşteri detayı) çıktıya yazılmadı (CLAUDE.md güvenlik kuralı)
- [ ] Bu modülün kapsamı dışındaki bir karar varsa ilgili agent'a görev üretildi (kendi başına karar verilmedi)
