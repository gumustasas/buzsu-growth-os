# Skill: Google Search Console

## Amaç
Google Search Console verilerini okuyarak Buzsu.com.tr ve Suvesu.com için
arama performansı analizi, kapsam denetimi ve iyileştirme önerileri üret.

## İlgili Agent
`seo-agent`, `analytics-agent`

## Kapsam
- Performans raporu: tıklama, izlenim, CTR, ortalama konum
- Kapsam denetimi: indeksleme sorunları, dışlanan URL'ler
- Zenginleştirilmiş sonuç hataları (şema sorunları)
- Sayfa deneyimi raporu (Core Web Vitals, HTTPS, mobil uyumluluk)
- AI Overview izlenimlerinin takibi

## Kurallar
Bu skill yalnızca GSC verisi okur ve rapor üretir. Ayar değişikliği yapmaz.

## Onay Notu
Veri okuma = SAFE PATCH. Aksiyona dönüştürme = ilgili skill onay süreci.
