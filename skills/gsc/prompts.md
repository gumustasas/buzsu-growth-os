# GSC Analiz Görev Promptları

## 1. Aylık Performans Raporu
```
GSC'den son 28 günlük performans verisini çek.
Raporla: toplam tıklama, izlenim, CTR, ort. pozisyon.
Top 10 sorgu ve top 10 sayfa tablolarını hazırla.
Önceki 28 günle karşılaştır.
/drafts/gsc-performans-[YYYY-MM].md olarak kaydet.
```

## 2. CTR İyileştirme Analizi
```
Ortalama pozisyon 1-10 arasında ama CTR ortalamanın altında olan sayfaları listele.
Her sayfa için: mevcut başlık, mevcut meta açıklama, iyileştirme önerisi.
Maksimum 10 sayfa, tıklama potansiyeline göre sırala.
```

## 3. Kapsam Sorunu Raporu
```
GSC Kapsam raporundaki "Hata" ve "Uyarı" sayfalarını analiz et.
Her sorun türü için: sayfa sayısı, örnek URL'ler, olası neden, önerilen aksiyon.
/drafts/gsc-kapsam-[YYYY-MM].md olarak raporla.
```

## 4. Şema Hata Raporu
```
GSC Zenginleştirilmiş Sonuçlar raporundaki tüm hata ve uyarıları listele.
Şema türüne göre grupla (Product, FAQ, HowTo).
Her hata için: etkilenen URL sayısı, hata mesajı, düzeltme adımı.
```

> Görevleri /tasks altında task dosyası oluşturarak takip et.
