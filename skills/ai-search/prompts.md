# AI Search Optimizasyon Görev Promptları

## 1. AI Alıntılanabilirlik Analizi
```
Suvesu.com'daki [URL] sayfasını analiz et.
Perplexity veya ChatGPT'nin bu sayfayı alıntılamasını engelleyen
3 yapısal sorunu tespit et. Her sorun için somut düzeltme öner.
Ürün iddiası uydurma. Bulguları /drafts/ai-audit-[slug].md olarak kaydet.
```

## 2. Konuşma Sorgusu Optimizasyonu
```
"[konu] nedir / nasıl çalışır / neden önemli" formatında
10 adet Türkçe konuşma sorusu üret.
Her soru için BUZSU veya Suvesu içeriğinden yanıt taslağı yaz (max 60 kelime/yanıt).
Çıktı: /drafts/content/konusma-sorgulari-[konu].md
```

## 3. llms.txt Taslağı
```
Buzsu.com.tr için minimal bir llms.txt dosyası oluştur.
İçerik: site amacı, marka bilgisi, izin verilen içerik,
kısıtlanan içerik (müşteri verisi, fiyat listesi), 10 temel URL.
Max 30 satır. Fiyat veya teknik spec yazma.
/drafts/llms-txt-v1.md olarak kaydet.
```

## 4. AI Kaynaklı Trafik Raporu
```
GSC verisinden AI Overview veya referral olarak gelen trafik değişimini raporla.
Aylık kıyaslama tablosu oluştur.
Kayıp varsa nedenini ve aksiyon öner.
/drafts/gsc-ai-trafik-[YYYY-MM].md olarak kaydet.
```

> Görevleri /tasks altında task dosyası oluşturarak takip et.
