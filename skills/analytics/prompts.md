# Analytics Görev Promptları

## 1. Aylık Büyüme Raporu
```
GA4'ten son 30 günlük veriyi çek.
Raporla: oturum sayısı, WhatsApp tıklaması, dönüşüm oranı, top 10 trafik kaynağı.
Önceki ay ve yılın aynı dönemiyle karşılaştır.
/drafts/analytics-buyume-[YYYY-MM].md olarak kaydet. Müşteri PII yazma.
```

## 2. Dönüşüm Hunisi Analizi
```
Oturum → ürün sayfası → WhatsApp hunisini analiz et.
Her adımda drop-off oranını hesapla.
En yüksek drop-off noktası için 3 iyileştirme önerisi sun.
```

## 3. Kampanya Performans Raporu
```
[Kampanya UTM kaynağı] için son [N gün] performansını analiz et.
Tıklama, oturum, dönüşüm değerlerini tablola.
/drafts/kampanya-rapor-[tarih].md olarak kaydet.
```

## 4. Anomali Tespiti
```
Son 7 günlük günlük oturum sayısını çek.
28 günlük ortalamadan ±%30 sapan günleri işaretle.
Olası neden (algoritma güncellemesi, teknik sorun, kampanya) öner.
```

> Görevleri /tasks altında task dosyası oluşturarak takip et.
