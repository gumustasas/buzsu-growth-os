# MCP — Model Context Protocol Entegrasyonları — BUZSU Uygulama Adımları

## Adımlar

1. MCP ile ilgili tasarım kararlarını `tasks/automation/` altına not et.
2. Yeni MCP entegrasyonu önerisi `docs/` altına mimari karar olarak yazılır (mimari karar = MAJOR, onay gerekir).
3. Mevcut araçların (Airtable, n8n, GitHub, Vercel okuma) kullanımı bu skill'in checklist'ine göre denetlenir.

## Sınır / Onay Notu

Yeni MCP sunucusu/dış sistem bağlantısı CLAUDE.md tablosunda 'Dış sistem yazma bağlantısı' → MAJOR, onay gerekir.

## İlgili Dosyalar

- İlgili agent tanımı: `Yok — automation-agent dış sistem entegrasyonlarını kapsıyor; MCP-spesifik tasarım kararları bu skill'e referans verir.`
- Bu modülün kontrol listesi: `skills/mcp/checklist.md`
- Örnek promptlar: `skills/mcp/prompts.md`
