# MCP — Model Context Protocol Entegrasyonları — Örnek Promptlar

Bu promptlar, ilgili agent veya orchestrator'a doğrudan görev olarak verilebilir.

1. Mevcut mcp__n8n__ araç setinin Buzsu otomasyon ihtiyaçlarına uygunluğunu değerlendir.
2. Yeni bir Airtable yazma aracı önerilirse hangi onay adımlarından geçmesi gerektiğini belirle.
3. MCP araç şemalarında okuma/yazma ayrımının netliğini denetle.

## Görev Dosyası Formatı

Yukarıdaki promptlardan biri görev olarak verildiğinde, önce ilgili `tasks/<alan>/` klasöründe bir görev dosyası oluşturulur (yoksa), sonra `implementation.md`'deki adımlar izlenir.
