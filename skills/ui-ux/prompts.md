# UI/UX — Tasarım Sistemi ve Erişilebilirlik — Örnek Promptlar

Bu promptlar, ilgili agent veya orchestrator'a doğrudan görev olarak verilebilir.

1. Ürün sayfası CTA bölümünün WCAG AA kontrast uyumunu denetle.
2. Dashboard widget'larının 4 viewport genişliğinde responsive durumunu raporla.
3. Güven rozeti (garanti/sertifika) yerleşimi için tasarım önerisi hazırla.
4. Mevcut tasarımda agresif satış deseni (sahte aciliyet vb.) var mı tara.

## Görev Dosyası Formatı

Yukarıdaki promptlardan biri görev olarak verildiğinde, önce ilgili `tasks/<alan>/` klasöründe bir görev dosyası oluşturulur (yoksa), sonra `implementation.md`'deki adımlar izlenir.
