# UI/UX — Tasarım Sistemi ve Erişilebilirlik — BUZSU Uygulama Adımları

## Adımlar

1. Bulguları `tasks/cro/` altına (UI/UX bu görev alanıyla iç içe) ekle.
2. Tasarım/kod önerisini `drafts/code/<özellik>-ui.md` altına yaz.
3. Görsel kanıt (varsa ekran görüntüsü açıklaması) taslağa eklenir, gerçek kullanıcı verisi/PII içermez.

## Sınır / Onay Notu

Tasarım önerileri SAFE PATCH (taslak); uygulanması kod değişikliği olduğundan branch+PR, MAJOR.

## İlgili Dosyalar

- İlgili agent tanımı: `Yok — cro-agent ile yakın çalışır; tasarım kararları cro-agent'ın çıktısına girdi sağlar.`
- Bu modülün kontrol listesi: `skills/ui-ux/checklist.md`
