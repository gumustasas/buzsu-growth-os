# Teknik SEO Görev Promptları

## 1. Teknik Denetim Raporu
```
Buzsu.com.tr için teknik SEO denetimi yap.
Kontrol et: canonical, robots.txt, sitemap, hreflang, 4xx, Core Web Vitals, AI direktifleri.
Her sorun için: url, sorun türü, öncelik (P1/P2/P3), düzeltme adımı.
/drafts/technical-seo-audit-[tarih].md olarak raporla.
```

## 2. Core Web Vitals İyileştirme Planı
```
PageSpeed Insights verisiyle LCP, INP, CLS sorunlarını listele.
Her metrik için: mevcut değer, hedef değer, önerilen düzeltme.
Mobil öncelikli. Teknik olmayan açıklama da ekle.
```

## 3. Sitemap Denetimi
```
Mevcut XML sitemap'i kontrol et.
Eksik URL, dead link, wrong priority veya lastmod hatası var mı?
Güncellenmiş sitemap taslağını /drafts/sitemap-v[n].xml olarak kaydet.
```

## 4. Yönlendirme Haritası
```
Sitedeki tüm 301/302 yönlendirmelerini listele.
Zincir yönlendirme (A→B→C) ve döngü (A→B→A) var mı?
Düzeltme önerisini tablo formatında ver.
```

> Görevleri /tasks altında task dosyası oluşturarak takip et.
