# Teknik SEO — Uygulama Adımları

## Adımlar

1. **Crawl denetimi** — GSC Kapsam raporu + site taraması.
2. **CWV ölçümü** — PageSpeed Insights ile LCP/INP/CLS değerlerini al.
3. **Sorun listesi** — Öncelik sıralamalı (P1 kritik, P2 önemli, P3 iyileştirme) tablo oluştur.
4. **Taslak öneriler** — Her sorun için /drafts/technical-seo-[konu].md hazırla.
5. **Canonical/sitemap taslağı** — Gerekli güncellemeler /drafts altına.
6. **Onay** — Taslaklar insan onayına sunulur.
7. **Uygulama** — Onaylanan değişiklikler developer tarafından uygulanır.
8. **İzleme** — 2 hafta sonra GSC kapsam + CWV değerlerini tekrar ölç.

## Onay Notu
Sunucu tarafı değişiklikler (redirect, server config) MAJOR sınıf; developer + insan onayı zorunlu.
