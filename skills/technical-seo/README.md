# Skill: Teknik SEO

## Amaç
Buzsu.com.tr ve Suvesu.com'un teknik SEO altyapısını güçlendir:
taranabilirlik, indekslenebilirlik, site hızı ve Core Web Vitals.

## İlgili Agent
`seo-agent`, `schema-agent`

## Kapsam
- Crawl bütçesi optimizasyonu (robots.txt, sitemap, canonical)
- Core Web Vitals: LCP, INP, CLS ölçüm ve iyileştirme
- Hreflang yapısı (tr-TR için doğru etiketleme)
- 4xx/5xx hata tespiti ve düzeltme
- JavaScript rendering sorunları
- AI crawler direktifleri

## İncelenen Kavram Kaynakları
- Google Search Central teknik SEO yönergeleri
- web.dev Core Web Vitals kılavuzu

## Onay Notu
robots.txt, sitemap veya canonical değişikliği MINOR; sunucu yapılandırması MAJOR sınıf.
