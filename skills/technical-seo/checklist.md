# Teknik SEO Kontrol Listesi

## Taranabilirlik
- [ ] robots.txt doğru mu? Önemli URL'ler engelleniyor mu?
- [ ] XML sitemap güncel mi ve GSC'ye gönderildi mi?
- [ ] Canonical URL'ler tutarlı mı?
- [ ] Parametre kirliliği var mı? (UTM, session ID indekse girmiş mi?)
- [ ] AI bot direktifleri tanımlı mı?

## İndekslenebilirlik
- [ ] GSC Kapsam: "Dizine alındı" sayısı stabil mi?
- [ ] "Dizine alınmadı - tarandı" sayfaları analiz edildi mi?
- [ ] Noindex etiketleri yanlış sayfada yok mu?
- [ ] Hreflang (tr-TR) doğru uygulandı mı?

## Core Web Vitals
- [ ] LCP < 2.5 sn mi? (mobil öncelikli)
- [ ] INP < 200 ms mi?
- [ ] CLS < 0.1 mi?
- [ ] PageSpeed Insights skoru > 70 mi? (mobil)

## Bağlantı Sağlığı
- [ ] 4xx hata sayfası < 10 adet mi?
- [ ] Dahili kırık bağlantı yok mu?
- [ ] Yönlendirme zinciri max 1 hop mu?

## Genel BUZSU Kuralları
- [ ] Değişiklik sınıfı belirlendi mi?
- [ ] /drafts üzerinden onay alındı mı?
- [ ] Task dosyası açıldı mı?
