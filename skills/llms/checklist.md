# LLM Crawlability Kontrol Listesi

## llms.txt
- [ ] /llms.txt dosyası site kökünde mevcut mu?
- [ ] Dosya: site amacı ve marka bilgisi tanımlıyor mu?
- [ ] İzin verilen içerik belirtildi mi?
- [ ] Kısıtlı içerik (müşteri verisi, ham fiyat listesi) açıkça belirtildi mi?
- [ ] Önemli URL'ler (ürün kategorileri, SSS sayfası) listede mi?

## robots.txt
- [ ] GPTBot direktifi tanımlı mı?
- [ ] ClaudeBot direktifi tanımlı mı?
- [ ] PerplexityBot direktifi tanımlı mı?
- [ ] Google-Extended direktifi tanımlı mı?

## İçerik Yapısı
- [ ] Her sayfa bağımsız paragraflarla bölünmüş mü?
- [ ] Başlık hiyerarşisi H1>H2>H3 sırasını koruyor mu?
- [ ] Yanlış teknik bilgi veya fiyat bilgisi var mı? (0 olmalı)
- [ ] Ortalama cümle uzunluğu < 20 kelime mi?

## Hallüsinasyon Testi
- [ ] ChatGPT'de BUZSU sorgusu yapıldı mı?
- [ ] Perplexity'de BUZSU sorgusu yapıldı mı?
- [ ] Yanlış bilgi tespit edildi mi? (var/yok)

## Genel BUZSU Kuralları
- [ ] Müşteri verisi içerik dosyasına yazılmadı mı?
- [ ] Değişiklik onay sürecinden geçti mi?
- [ ] Task dosyası oluşturuldu mu?
